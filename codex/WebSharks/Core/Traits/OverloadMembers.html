Parameters

array|object $properties Property(s).
bool $writable Overloaded properties are writable?

Internal Note

Objects are passed by reference, so we can overload properties by reference. This only works if each property in the object does not yet exist in this class. Object properties can be set as writable or not writable.
Property names in an array can be overloaded; e.g., to provide public or JSON access. This only works if each property in the array already exists in this class. Overloaded object property names can be set as writable or not writable.
